# 斩杀线（Kill-line）计算器示例

这是一个基于 React + Vite 的小项目，用来演示如何计算个人的“斩杀线”（基于支出估算的财富安全线）以及抗风险能力评估。

快速开始：

```bash
cd /workspaces/kill-line-demo
npm install
npm run dev
```

主要想法：
- 斩杀线采用常见的 25x 年支出（4% 规则）作为参考。
- 抗风险能力使用流动性（月可支撑天数）、债务/收入比（DTI）和净资产等指标组合计算。

可扩展方向（建议）：
- 支持更详细的税务/profession 模拟
- 导入银行/投资数据以自动填充资产
- 添加情景模拟（失业、重大医疗支出）
- 支持美国各州税率与成本差异
